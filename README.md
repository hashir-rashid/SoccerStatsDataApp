# Soccerats Soccer Stats
Since I'm the coolest, we are using this name for the project and no objections. How to run:

## How to run
### Clone the repo and install dependencies
```bash
git clone https://github.com/hashir-rashid/SoccerStatsDataApp.git
cd backend
npm install
```

### Download the database file  
Go to https://www.kaggle.com/datasets/hugomathien/soccer, and place it in the `backend/database/` directory.  
Ensure it is named `database.sqlite`

### Run the app
```node
npm start
```
